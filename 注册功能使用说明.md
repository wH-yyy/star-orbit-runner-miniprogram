# 注册功能云函数使用说明

## 📦 已创建的云函数

### 1. sendVerificationCode（发送验证码）
**路径**: `cloudfunctions/sendVerificationCode/`

**功能**: 
- 发送6位数字验证码到用户手机
- 防重复发送（60秒内只能发送一次）
- 验证码有效期10分钟
- 自动存储到数据库

**调用示例**:
```javascript
wx.cloud.callFunction({
  name: 'sendVerificationCode',
  data: { phone: '13800138000' }
})
```

### 2. register（用户注册）
**路径**: `cloudfunctions/register/`

**功能**:
- 验证所有注册信息
- 检查学号和手机号是否已注册
- 验证验证码是否正确
- 密码加密存储（SHA256）
- 自动创建用户记录

**调用示例**:
```javascript
wx.cloud.callFunction({
  name: 'register',
  data: {
    phone: '13800138000',
    studentId: '1234567890',
    name: '张三',
    gender: '男',
    campus: '兴庆校区',
    className: '软件2201',
    college: '彭康书院',
    code: '123456',
    password: 'Password123'
  }
})
```

### 3. checkDuplicate（检查重复）
**路径**: `cloudfunctions/checkDuplicate/`

**功能**:
- 实时检查学号或手机号是否已被注册
- 提供即时反馈，优化用户体验

**调用示例**:
```javascript
// 检查学号
wx.cloud.callFunction({
  name: 'checkDuplicate',
  data: {
    type: 'studentId',
    value: '1234567890'
  }
})

// 检查手机号
wx.cloud.callFunction({
  name: 'checkDuplicate',
  data: {
    type: 'phone',
    value: '13800138000'
  }
})
```

## 🗄️ 需要创建的数据库集合

### 1. Users（用户表）
需要在云开发控制台创建集合 `Users`，字段如下：

```javascript
{
  _id: "自动生成",
  stu_id: "学号（字符串，10位）",
  name: "姓名",
  gender: "性别（男/女）",
  campus: "校区",
  class: "班级",
  college: "书院",
  phone: "手机号",
  password: "密码（SHA256加密）",
  openid: "微信OpenID",
  createTime: "注册时间",
  updateTime: "更新时间",
  status: "账户状态（active/banned）",
  totalDistance: "总跑步距离（米）",
  totalDuration: "总跑步时长（秒）",
  totalCount: "总跑步次数",
  avatar: "头像URL"
}
```

**索引建议**:
- `stu_id`: 唯一索引
- `phone`: 唯一索引
- `openid`: 普通索引

### 2. VerificationCodes（验证码表）
需要在云开发控制台创建集合 `VerificationCodes`，字段如下：

```javascript
{
  _id: "自动生成",
  phone: "手机号",
  code: "验证码（6位数字）",
  used: "是否已使用（布尔值）",
  createTime: "创建时间（时间戳）",
  expireTime: "过期时间（时间戳）",
  usedTime: "使用时间（时间戳，可选）"
}
```

**索引建议**:
- `phone`: 普通索引
- `expireTime`: 普通索引

## 🚀 部署步骤

### 第一步：上传云函数

在微信开发者工具中：

1. 右键点击 `cloudfunctions/sendVerificationCode` → 上传并部署：云端安装依赖
2. 右键点击 `cloudfunctions/register` → 上传并部署：云端安装依赖
3. 右键点击 `cloudfunctions/checkDuplicate` → 上传并部署：云端安装依赖

### 第二步：创建数据库集合

1. 打开云开发控制台
2. 进入「数据库」
3. 创建集合 `Users`
4. 创建集合 `VerificationCodes`
5. 按照上面的建议添加索引

### 第三步：配置数据库权限

在云开发控制台 → 数据库 → 权限设置：

- **Users**: 
  - 仅创建者及管理员可读写（默认）
  - 或自定义权限规则
  
- **VerificationCodes**: 
  - 仅创建者及管理员可读写（默认）

### 第四步：测试功能

1. 在小程序中打开注册页面
2. 输入手机号，点击"获取验证码"
3. 查看控制台输出的验证码（开发环境）
4. 填写完整信息，提交注册
5. 检查数据库中是否创建了用户记录

## ⚠️ 重要说明

### 开发环境 vs 生产环境

**当前代码在开发模式下**会在控制台输出验证码，方便测试。

**上线前必须做的修改**：

1. **删除测试验证码输出**
   
   在 `sendVerificationCode/index.js` 中删除这两行：
   ```javascript
   console.log('验证码（仅供测试）:', code)  // 删除
   devCode: code  // 删除此行！
   ```

2. **接入真实短信服务**
   
   在 `sendVerificationCode/index.js` 中取消注释短信服务代码，并配置：
   - 腾讯云短信：https://cloud.tencent.com/product/sms
   - 阿里云短信：https://www.aliyun.com/product/sms
   - 其他短信服务商
   
   需要配置：
   - SecretId / SecretKey
   - 短信签名
   - 短信模板ID

### 密码安全

当前使用 SHA256 加密密码，这是基础的安全措施。

**更安全的方案**（推荐生产环境）：
```javascript
// 安装 bcrypt
npm install bcryptjs

// 在云函数中使用
const bcrypt = require('bcryptjs')
const hashedPassword = bcrypt.hashSync(password, 10)
```

## 🔧 可选优化

### 1. 添加实时验证

在注册页面添加失焦验证，提升用户体验：

```javascript
// 学号失焦时验证
onStudentIdBlur() {
  const studentId = this.data.studentId
  if (studentId && studentId.length === 10) {
    wx.cloud.callFunction({
      name: 'checkDuplicate',
      data: { type: 'studentId', value: studentId }
    }).then(res => {
      if (res.result.data.exists) {
        wx.showToast({
          title: '该学号已被注册',
          icon: 'none'
        })
      }
    })
  }
}
```

### 2. 添加图形验证码

防止恶意刷验证码，可以在发送短信验证码前先验证图形验证码。

### 3. 限制注册频率

在云函数中添加IP或设备限制，防止批量注册。

## 🐛 常见问题

**Q: 验证码发送失败？**
A: 检查云函数是否正确上传，数据库集合是否创建。

**Q: 提示"该学号已被注册"但确实没注册过？**
A: 检查数据库 Users 集合中是否有重复数据，可能是测试时创建的。

**Q: 密码验证不通过？**
A: 密码必须8-20位，包含字母和数字，可包含 @#$%&-_ 符号。

**Q: 如何清理过期的验证码？**
A: 可以创建一个定时触发器云函数，定期清理过期数据。
